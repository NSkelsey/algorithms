algorithms
==========

Implementations of various algorithms in various languages. Enjoy


power.c
-------
Computes a^n in O(logn) operations.

example:

```bash
./a.out 4 12
```


select.py
---------
Selects the kth ordered element in an unordered list in O(n) time.

Example usage:
```
bleh
```

Which outputs:
```
blah
```
